# 前台首页改版说明（V3）— 可直接发给客户

---

## 您会看到的变化

### 顶部

- **Logo 与语言**：站 Logo 和语言切换（中文/英文）移到页面中间，更醒目。
- **跑马灯**：Logo 下方有一条全宽公告条，文字自动滚动，内容由后台配置。

### 中间主区域

- **左侧**：大图轮播展示活动（Promotion），可左右滑动；点某张图会在页面内展开该活动详情，不会跳走或跳回第一张。
- **中间**：预留一块大图位，后续可替换为合作图/广告图（后台上传即可）。
- **右侧**：最新交易列表（Live Transaction），**一直向上滚动**，带细线分隔，看起来整齐；每条只显示手机尾 4 位和金额，Deposit 不显示入金方式，Withdraw 显示游戏商名称。

### 登录与操作区（在交易列表下方）

- **未登录**：显示 **Login**、**Register** 按钮。
- **已登录**：显示 **余额**、**Deposit（入金）**、**Withdraw（出金）**、**Refresh（刷新）**、**Sign out（登出）**。
- Deposit 按钮为蓝色、Withdraw 为金色（颜色可在后台微调）；这些按钮以后也可换成图片（后台上传）。

在 Deposit/Withdraw 旁边会显示最低/最高入金与出金金额（由后台配置）。

---

## 后台可配置内容

- 跑马灯公告文案（可多条）
- 活动大图与内容
- 中间插图槽的大图
- Partnership 圆形徽章图
- 登录/注册/Deposit/Withdraw/刷新/登出等按钮的图片（可选）及 Deposit/Withdraw 颜色
- 最低/最高 Deposit、Withdraw 金额

---

## 移动端

- 跑马灯、Logo、语言仍保留；中间区域改为上下排列：大图轮播 → 插图槽 → 交易滚动列表 → 登录/操作区，保证不挤、易点。
