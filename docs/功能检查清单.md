## 五、Settings 子页（config/settings-nav.ts）

- **Theme**：/admin/settings/theme → ThemeSettingsClient，保存至 GET/PATCH /api/admin/theme
- **Frontend**：general、notices、popups、download-bar、home-media、display 均渲染 **ThemeSettingsClient**，与 Theme 同源保存
- **Promotion Center**：list（PromotionListSettingsClient）、content(→list)、media/links/layout/preview（带 promotionId 跳编辑，否则引导回 list）
- **Referral Center**：general、sharing、**display** 均用 **AdminReferralSettingsClient**，保存至 /api/admin/settings/referral
- **Finance**：banks(→bank)、deposit-rules(→deposit-topup-rules)、withdraw-rules、payment-gateways(→payment-gateway)
- **Integrations**：游戏管理 /admin/settings/game-providers（API+Logo 合并）、game-api/games→game-providers、whatsapp(→settings/whatsapp)
- **Account & Security**：profile、password、security（2FA+IP 白名单+Create Admin 入口）、login-history、admins

以上 href 均有对应 page 或 redirect，且功能已连接（无占位「comingSoon」断链）。
