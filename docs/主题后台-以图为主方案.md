# 主题后台：以照片为主、颜色收起

## 一句话结论
用户不会用颜色来设计前台美观，只会用照片。后台改为「以图为主」：主界面只保留风格预设 + 各类图片/照片设置；所有颜色与尺寸收起到「高级」折叠区，可删或保留给少数人用。

## 可以删掉颜色吗？
- **主界面**：可以不再展示颜色设置，普通用户只看到「整体风格 + 整页背景图 + Logo + 轮播/合作方/徽章/按钮图」等。
- **数据与前台**：建议**不删** theme 里的颜色字段，只从后台 UI 隐藏或收起。这样：
  - 已有站点已保存的颜色不会丢；
  - 前台仍可用默认色或设计预设的配色；
  - 若以后要恢复「调颜色」只要再打开高级即可。

## 后台结构建议（以图为主）

1. **整体外观**
   - 设计风格（默认/简约/奢华/电竞/柔和）
   - 整页背景图（只保留「图」，整页背景色移到高级）

2. **站点与入口**
   - 站名、Logo 图、登录/注册/充值/提款/客服链接（不变）

3. **顶栏与跑马灯**
   - 跑马灯单条/多条文案
   - （可选）顶栏背景图、跑马灯区域背景图（若加字段）
   - 顶栏/跑马灯颜色 → 移到高级

4. **首页结构**
   - 区块标题、路径、游戏分类、Pills（不变）

5. **弹窗与条**
   - 年龄验证、下载条（标题/副标题/按钮/图片）（不变）

6. **操作栏**
   - 存款/提款按钮**图片**、限额文案（不变）
   - 存款/提款**颜色** → 移到高级

7. **首页内容**
   - 轮播图、合作方、快捷入口（图+链接）、底部导航（图+链接）（不变）

8. **其它图片**
   - 合作徽章、中间插槽、次级横幅、流水区背景图（不变）
   - 整页背景色 → 移到高级

9. **展示样式**
   - 优惠列表样式、社交样式、优惠字体预设（不变，非颜色）

10. **促销卡片**
    - 图片高度、列数、显示内容（不变）

11. **高级：颜色与尺寸**（折叠，默认收起）
    - 主题主色/强调色
    - 顶栏/跑马灯颜色
    - 操作栏按钮颜色
    - 手机端 Vivid 颜色与尺寸
    - 桌面版颜色与尺寸
    - 流水表/推荐区块/客服栏颜色
    - 前台语义色与字体（或只保留字体/字号，颜色放进高级）

## 实施步骤
1. 整体外观：只保留「设计风格」+「整页背景图」，删掉主色/强调色/整页背景色输入框。
2. 顶栏与跑马灯：删掉顶栏背景/边框、跑马灯背景/边框/文字色输入框。
3. 操作栏：删掉充值/提款颜色输入框，保留限额与按钮图片。
4. 删掉整块「手机端颜色与尺寸」「桌面版颜色与尺寸」「流水表/推荐/客服颜色」「前台语义色与字体」的**主界面展示**。
5. 新增一栏「高级：颜色与尺寸」，默认折叠，内放以上所有颜色与尺寸字段；保存/读取逻辑不变。

这样用户平时只看到「风格 + 各种图片」，颜色功能算删除出主界面，需要时仍可在高级里调。
